思科easyVPN的配置

By Songxin at: 2018-12-26
路由技术 VPN 接入技术

配置步骤

1. 创建IKE策略集，该策略集至少要能与vpnclient的一个内置策略集相匹配，以便在server和client之间建立IKESA 
```
crypto isakmp policy 10
authentication pre-share
encryption 3des
group 2
```
2. 定义要推送给client的组属性，其中包含分配给client的地址池、 pre-share key等
```
ip local pool remote-pool 12.0.0.100 12.0.0.200
crypto isakmp client configuration group vpngroup
key 123456
pool remote-pool
aaa new-moduel
aaa authorization network vpn-remote-access local```
3. 定义IPSec变换集（只用于client触发建立IPSecSA时，如果是server触发建立IPSecSA就不需要使用）
```
Router0(config)#crypto ipsec transform-set ipsectrans esp-3des esp-md5-hmac 
Router0(config)#crypto dynamic-map dynamic-zsx 1
Router0(config-crypto-map)#set transform-set ipsectrans
Router0(config-crypto-map)#reverse-route 
Router0(config-crypto-map)#exit
Router0(config)#crypto map zsx 1 ipsec-isakmp dynamic dynamic-zsx 
```
4. 启用DPD死亡对端检测

5. 配置Xauth扩展认证
```
Router0(config)#aaa authentication login vpnuser local
Router0(config)#username zsx password 2018200967
Router0(config)#crypto map zsx client authentication list vpnuser
```
6. 把cryptomap应用到路由器端口上
```
Router0(config)#interface g0/0/0
Router0(config-if)#crypto map zsx
*Jan 3 07:16:26.785: %CRYPTO-6-ISAKMP_ON_OFF: ISAKMP is ON
Router0(config-if)#exit
```
